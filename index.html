Проекты

Task Management System
https://github.com/sergeloie/TaskManagementSystem

Бекэнд веб-сервиса для управления задачами. Позволяет регистрироваться пользователям, создавать и редактировать задачи, назначать исполнителей, оставлять комментарии. Написан на Java 21 с использованием Spring Framework. Авторизация через JWT с использованием Spring Security. Реализована возможность запуска через Docker Compose совместно с Postgres.

Веб-сервис склоняет числительные от 0 до 999_999_999_999 включительно, на русском языке по падежам и родам. Написан на Java 21 с использованием Spring Web. Сервис упакован в Docker и развёрнут на Render.com

Счётчик IP адресов
https://github.com/sergeloie/IP-Addresses-Counter

Утилита для подсчёта количества уникальных IP адресов в файле. Применение битовых массивов позволило использовать ограниченный объём памяти, независимо от размера файла. При разработке использовался Java Microbenchmark Harness, для сравнения быстродействия различных методов.


Сервис сокращения ссылок
https://github.com/sergeloie/URL_Shortener

Бекэнд для сервиса сокращения ссылок. Написан на Java 21 с использованием Spring Framework. Для кеширования запросов и построения рейтинга популярности ссылок применяется Redis, основная БД для хранения - PostgreSQL. Приложение и базы данных разворачиваются из образов Docker с помощью Docker Compose. Для ускорения разработки и повышения качества кода применяется Amplicode.
