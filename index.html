Проекты

    Для отработки навыков я реализовал несколько самостоятельных проектов.

  • URL Shortener
    в разработке
    Бекэнд для сервиса сокращения ссылок. Написан на Java 21 с использованием Spring Framework. Для кеширования запросов и построения рейтинга популярности ссылок применяется Redis, основная БД для хранения - PostgreSQL. Приложение и базы данных разворачиваются из образов Docker с помощью Docker Compose. Для ускорения разработки и повышения качества кода применяется Amplicode.

  • IP Addresses counter
    завершён, планируется переработка с использованием многопоточности
    Утилита для подсчёта количества уникальных IP адресов в файле. Основная идея заключается в том, что IP адрес представляется в виде числа и информация о его наличии хранится в BitSet на соответствующей позиции. Этот подход показал существенно меньший расход памяти, по сравнению в префиксными деревьями. Для замера производительности различных алгоритмов работы применялся Java Microbenchmark Harness.
